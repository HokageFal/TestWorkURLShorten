# URL Shortener API

Тестовое задание на позицию Middle Backend Developer.

## Стек
- Python 3.12+
- FastAPI
- SQLite3 (без ORM)

## Запуск
1. Создайте виртуальное окружение: `python -m venv venv`
2. Активируйте его: `source venv/bin/activate` (или `venv\Scripts\activate` на Windows)
3. Установите зависимости: `pip install .`
4. Запустите сервер: `uvicorn src.main:app --reload`
